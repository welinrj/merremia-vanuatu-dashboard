VCAP2 GIS Data Portal

A web-based geographic information system for the Vanuatu Climate Adaptation Project 2 (VCAP2). Upload, visualize, and manage marine conservation data across Vanuatu's provinces. Powered by GitHub for secure, version-controlled data storage.

🌍 Interactive Map

Explore conservation sites on an interactive Leaflet map with dark, satellite, and street basemaps. Click features to view detailed attributes and photos.

📂 File Upload

Upload GeoJSON, KML, and Shapefile (.zip) files directly in your browser. Data is parsed client-side and displayed instantly on the map.

📈 Analytics Dashboard

Real-time charts and statistics showing dataset coverage, biodiversity indices, area distribution, and conservation status breakdown.

📄 Attribute Tables

View and explore feature attributes in sortable tables. Download datasets as GeoJSON for use in QGIS, ArcGIS, or other GIS software.

🔒 GitHub Storage

All datasets stored in your GitHub repository with full version control. Share data with team members via repository access.

🌎 Vanuatu Focused

Pre-configured for Vanuatu's marine conservation areas with sample data from Shefa Province including MPAs, sanctuaries, and conservation zones.

Quick Start Guide

  1. Open the Map Portal to view pre-loaded marine conservation areas
  2. Click on any feature polygon to see its attributes (area, status, biodiversity index)
  3. Use the Upload zone in the sidebar to add your own GeoJSON or KML data
  4. Click Table on any dataset to view its attribute table
  5. Visit the Dashboard for charts, statistics, and the coverage overview map
  6. Use Download to export any dataset as GeoJSON for desktop GIS tools
